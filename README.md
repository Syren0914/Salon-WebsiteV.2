
## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `VITE_SUPABASE_URL` in [.env.local](.env.local) to your Supabase Database URL
3. Set the `VITE_SUPABASE_SERVICE_ROLE_KEY` in [.env.local](.env.local) to your  Supabase Database Service Role Key
4. Set the `VITE_SUPABASE_ANON_KEY` in [.env.local](.env.local) to your Supabase Database Anon key
 
5. Run the app:
   `npm run dev`
